# Conky config

Conky configuration based on the Mimosa theme from <https://github.com/closebox73/Leonis/>

Latest working GNOME version: 45.0

Required packages:

- conky-lua-nv
- playerctl

Required fonts:

- Noto Nerd Font
- feather
- Symbols Nerd Font
- Font Awesome 6 Brands

Fonts need to be located in `/home/<username>/.fonts`.
